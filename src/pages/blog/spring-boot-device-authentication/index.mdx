Security is a big deal nowadays.
Data is getting the most valuable thing and more and more sensitive information are stored open to the naked eye.
And although your security mechanisms might be superior, the users password might not be. So what can you do to protect the user? A possible way to
make your users accounts more safe, is device authentication.

We will cover device authentication in this post and build an example application with
[Spring Boot](https://spring.io/).
